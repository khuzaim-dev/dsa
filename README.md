## How to Run

Compile a Java file:

```bash
javac src/<package>/<folder>/<FileName>.java
```

Run a Java file:

```bash
java -cp src <package>.<folder>.<FileName>
```

Example:

```bash
javac src/arrays_strings/brute/FindClosestNumberToZeroBrute.java

java -cp src arrays_strings.brute.FindClosestNumberToZeroBrute
```
